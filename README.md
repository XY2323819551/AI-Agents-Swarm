# Swarm Demo

基于OpenAI API的多Agent系统演示项目。

## 项目结构

```
.
├── examples_swarm/
├── llm_pool/
├── poc_swarm/
├── tool_pool/
└── utils/
```

## 功能特性

- 多Agent系统
- 工具调用
- Agent切换
- 上下文管理

## 使用方法

1. 安装依赖
```bash
pip install -r requirements.txt
```

2. 运行示例
```bash
python poc_swarm/poc_swarm_stage6.py
```
